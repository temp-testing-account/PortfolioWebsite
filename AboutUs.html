About Us

We at Tech Nerds, are computer programmers who specialize in website design. Our duties include determining the structure and design of web pages, striking a balance between functional and aesthetic design and ensuring web design is optimized for smartphones. We are responsible for implementing visual elements that users see and interact with in a web application.

Our Responsibilities

  • Determining the structure and design of web pages.
  • Ensuring user experience determines design choices.
  • Developing features to enhance the user experience.
  • Striking a balance between functional and aesthetic design.
  • Ensuring web design is optimized for smartphones.
  • Building reusable code for future use.
  • Optimizing web pages for maximum speed and scalability.
  • Utilizing a variety of markup languages to write web pages.
  • Maintaining brand consistency throughout design.
  • SEO optimized web pages

Our Skillsets

  • Understanding of key design principles.
  • Proficiency with HTML, CSS, JavaScript and jQuery.
  • Experience with graphic design applications such as Adobe Photoshop and Illustrator.
  • Experience with responsive and adaptive design.
  • Familiar with UI layouts, Bootstrap, and the CSS Grid system
  • Understanding of SEO principles.
  • Excellent verbal communication skills.
  • Good interpersonal skills.
  • Customer Satisfaction
